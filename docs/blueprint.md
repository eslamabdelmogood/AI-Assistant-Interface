# **App Name**: Green Assistant

## Core Features:

- AI-Powered Chat Interface: Enable engineers to interact with the Green Box network using natural language (text or voice).
- Equipment Status Query: Allow users to query the current status of equipment through the chat interface.
- Maintenance Report Generation: Enable users to request and receive maintenance reports through the chat interface.
- Spare Parts Ordering: Allow users to order spare parts directly through the chat interface.
- Real-time Diagnostics: Analyze live sensor data and provide real-time diagnostics through the AI assistant, presenting information from multiple sensors or history through the LLM as needed; the LLM will act as a tool in this step.
- Predictive Maintenance Insights: Provide actionable insights based on machine learning predictions, displayed through the AI assistant.
- Dispatch Drone: Generates a work order and dispatches a drone for parts pickup

## Style Guidelines:

- Primary color: Deep teal (#008080) to evoke a sense of technological sophistication and reliability.
- Background color: Dark grayish-teal (#263238) for a professional, easy-on-the-eyes dark theme.
- Accent color: Electric lime (#B0FF14) to highlight key information and interactive elements.
- Body and headline font: 'Inter' sans-serif, to maintain a modern and objective feel throughout the application.
- Use clean, outlined icons to represent equipment status, maintenance actions, and data visualizations.
- A split-screen layout, to simultaneously display the chat interface, and also equipment data visualizations
- Use subtle transitions and animations to provide feedback to user actions, such as loading data or submitting orders.
